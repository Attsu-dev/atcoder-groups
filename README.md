# AtCoder Groups

AtCoder Groups は、AtCoderでの活動状況をグループ内で共有するためのWebアプリケーションです。

代表者がメンバーのAtCoderユーザIDを登録することで、ページが生成されます。

## 表示する情報

### ランキング

- レートランキング
- レート増加量ランキング（ウィークリー、マンスリー）
- point sum ランキング（ウィークリー、マンスリー）

### ニュース

- 色変通知（例：〇〇さんが入茶しました！）
- コンテスト初参加通知（例：〇〇さんが初めてコンテストに参加しました！）
- streak通知（例：〇〇さんが10日連続でACしています！）

### その他

- メンバーが最近解いた問題

## 開発

```bash
npm install
npm run dev
```

品質確認には以下を使います。

```bash
npm test
npm run lint
npm run build
```
